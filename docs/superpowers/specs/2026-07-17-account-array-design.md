# 账户数组示例设计

## 目标

使用 Java 数组存储五个账户对象，打印每个账户的信息，并根据账户余额显示状态。项目包含可自动运行的测试。

## 项目结构

项目使用 Maven 和 JUnit 5：

- `Account`：账户领域对象，保存账号、户名和余额。
- `AccountDemo`：创建并返回包含五个账户的 `Account[]`，并在程序入口中逐一打印。
- `AccountTest`：验证账户状态规则和五账户数组。

## 账户状态规则

- 余额大于 `0`：状态为 `active`。
- 余额等于或小于 `0`：状态为 `inactive`。

状态由 `Account.getStatus()` 根据当前余额动态计算，不额外存储，避免余额与状态不一致。

## 输出

每个账户输出一行，格式如下：

```text
Account{id='A001', owner='张三', balance=1000.0, status='active'}
```

## 测试

JUnit 5 测试覆盖：

1. 正余额账户返回 `active`。
2. 零余额账户返回 `inactive`。
3. 负余额账户返回 `inactive`。
4. 演示程序创建的数组长度为五，且五个元素均为账户对象。

## 验收标准

- `mvn test` 成功通过全部测试。
- 运行 `AccountDemo` 时恰好打印五行账户信息。
- 每行状态均符合余额判断规则。
